Welcome to BTO - Core Manual

A comprehensive guide to streamline and utilize the BTO Core web application effectively.

Introduction

BTO - Core is a robust web application designed to simplify and optimize the operations of Bilkent University's Information Office. From automating tour applications to managing data securely, BTO Core ensures efficiency, accuracy, and seamless communication.

Terms such as 'Users', 'Bilkent Information Staff', and 'BTO' are used interchangeably throughout this guide.

Core Features

1. Automated Tour Applications

High schools can submit campus tour applications using a user-friendly form. Applications are prioritized using a built-in algorithm for efficiency.

2. Role-Based Dashboards

Different roles such as coordinators, advisors, guides, and candidates have tailored dashboards to access relevant tools and data seamlessly.

4. Real-Time Communication

5. Feedback System

A streamlined mechanism to collect feedback from high schools, guides, and candidates, helping to monitor and improve performance.

Security

3-Layered Access Security

  • Frontend Layer: Unauthorized users are redirected to the login page to prevent access to protected content.
  • Backend Layer 1: JWT token protection guards against unauthorized external access.
  • Backend Layer 2: Role-based permissions ensure users only access the data they are authorized for.

These layers guarantee that sensitive information remains secure and the system operates smoothly.

Troubleshooting

Why am I unable to access certain pages?

Ensure you have the correct permissions for the page. If access issues persist, contact the system administrator.

How do I reset my password?

On the login page, click on the 'Forgot Password' link and follow the instructions sent to your email.

How do I report technical issues?

If you encounter any errors, contact the technical support team at agorateam@bilkent.edu.tr.
